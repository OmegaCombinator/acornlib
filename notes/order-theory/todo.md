# Order Theory and Lattices

Goal: provide a broad order-theoretic language that later algebra, topology, and analysis can share.

- [ ] Complete the preorder, partial order, and linear order APIs
- [ ] Add a strong library for `<`, `<=`, `min`, `max`, and interval manipulations
- [ ] Implement semilattices, lattices, and distributive lattices
- [ ] Add complete lattices with `sup`, `inf`, `sSup`, and `sInf`
- [ ] Develop monotone, antitone, and order-embedding APIs
- [ ] Add order isomorphisms and transport lemmas
- [ ] Support Galois connections, insertions, and coinsertions
- [ ] Add well-founded orders and induction/recursion principles
- [ ] Build order-theoretic fixed point results
- [ ] Connect order theory cleanly with algebraic and topological structures
