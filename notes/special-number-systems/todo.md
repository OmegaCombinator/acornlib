# Special Number Systems

Goal: extend the core scalar tower beyond the basic ordered fields and complexes currently in view.

- [ ] Expand the complex-number API for algebra and analysis
- [ ] Add extended real numbers with infinities
- [ ] Support nonnegative reals and related ordered scalar systems
- [ ] Add p-adic numbers
- [ ] Add p-adic integers and valuation-oriented lemmas
- [ ] Support algebraic numbers as a reusable scalar type
- [ ] Add computable-real or constructive-real infrastructure if useful
- [ ] Consider hyperreal or nonstandard-analysis foundations
- [ ] Add standard coercions and comparison lemmas between number systems
- [ ] Normalize notation and transport lemmas across the scalar tower
