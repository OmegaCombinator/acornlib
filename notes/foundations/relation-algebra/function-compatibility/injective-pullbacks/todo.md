# Injective Pullbacks

Goal: capture the extra equalities and order-style transport theorems that become true when a pullback map is injective.

- [ ] Prove that the pullback of equality along an injective function is equality
- [ ] Prove that injective pullback preserves antisymmetry
- [ ] Prove that injective pullback preserves partial-order style packages built from reflexive, transitive, and antisymmetric fields
- [ ] Prove that injective pullback commutes with reflexive closure
- [ ] Decide whether strict-order pullback lemmas should live here or in order theory
- [ ] Prove the clean strict-order pullback theorem if that decision stays here
- [ ] Add the equality-pullback theorem as a reusable helper for later closure proofs
- [ ] Refactor one proof to use equality pullback instead of a local injectivity argument
- [ ] Audit whether subgroup/order-style code already has a hidden use for these lemmas
- [ ] Collapse this leaf once equality and reflexive-closure pullback are in place and used once
